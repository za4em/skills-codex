# Codex Skills

This repo now contains helper skills.

## Included Skills

- `$commit`
  Create or update conventional commit messages and commit cleanly.
- `$branch-description`
  Generate fixed-format Github/GitLab PR/MR descriptions from a branch commit range.
- `$caveman`
  Ultra-compressed communication mode that cuts token usage while keeping technical accuracy.
- `$figma-implement`
  Implement UI from Figma nodes or URLs with close visual fidelity.
- `$visualize`
  Create a self-contained HTML review page for a PR, local changes, or user-specified scope.
